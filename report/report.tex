\title{A Collection of Benchmark Problems for the Sabre Narrative Planner}

\author{Stephen G. Ware and Rachelyn Farrell}

\date{\today}

\abstract{
In order to test the Sabre Narrative Planner, we have compiled a collection of
story planning problems from several authors. This report identifies the origins
of these problems, describes the modifications we made so they can run in Sabre,
and gives recommended settings when searching for solutions.
}
